📌 MVC MİMARİSİ:
   • Model: Veritabanı işlemleri (app/models/)
   • View: Kullanıcı arayüzü (app/views/)
   • Controller: İş mantığı (app/controllers/)
   • Router: URL yönlendirme (app/router.php)

🔒 GÜVENLİK:
   • CSRF koruması tüm formlarda aktif
   • XSS koruması output filtreleme ile sağlanır
   • SQL Injection koruması PDO prepared statements ile
   • Oturum güvenliği HTTP-only cookies ile

🎯 MULTI-TENANT YAPI:
   • Her kurum (COVE) için ayrı veri izolasyonu
   • cove_id ile filtreleme zorunlu
   • SuperAdmin tüm COVE'lere erişebilir

🚀 PERFORMANS:
   • Query cache sistemi (cache/queries/)
   • Asset minification (composer run minify)
   • Autoloader optimizasyonu aktif

📊 VERİTABANI:
   • PDO ile bağlantı
   • UTF-8 karakter seti
   • Transaction desteği
   • Migration sistemi hazır

🛠️ DEVELOPMENT:
   • PHPStan Level 2 statik analiz
   • PHPUnit test framework
   • Composer autoloading
   • .env dosyası ile yapılandırma
